# swiggy_dashboard (Interactive Dashboard Creation using MS Excel)

I developed a comprehensive project in Excel, creating multiple and tables and charts to analyze the data. This process involved several stages, including data preprocessing, data cleaning, and data visualization.

## Project Objective

To create an interactive sales and Trend analysis dashboard to understand business performance, customer purchasing behavior, and product trends. This project helps identify important insights that can improve sales strategies and Trends.

## Process

* Collected and cleaned raw sales data in Excel.
* Checked for missing values, duplicate records, and formatting issues.
* Standardized data types and organized the dataset properly.
* Created pivot tables and pivot charts for analysis.
* Designed an interactive dashboard using slicers and filters.
* Added visualizations to make the dashboard user-friendly and dynamic.

* Analyzed Swiggy sales performance using interactive Power BI visualizations.

* Compared Veg vs Non-Veg food sales contribution.
* Evaluated state-wise sales distribution across India.
* Tracked important KPIs such as Total Sales, Average Rating, Average Order Value, and Total Orders.
* Highlighted top-performing cities and restaurants based on sales data.
